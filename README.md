# Bangumi Episodes Comments

番剧剧集评论数据集，按年份和季度组织。

## 数据来源

评论数据来源于 [Bangumi 番组计划](https://bgm.tv/)，通过爬虫程序收集各番剧每集的用户评论。

## 目录结构

```
comments_text/
├── 1979/
│   └── Q2/
│       └── 50.txt
├── 1982/
├── 1985/
├── ...
└── 2025/
```

- 第一层目录为**年份**（1979-2025）
- 第二层目录为**季度**（Q1-Q4），部分年份可能只有单个季度
- 文件以番剧 ID 命名（`.txt` 格式）

## 文件格式

每个 `.txt` 文件包含该番剧所有剧集的用户评论，格式如下：

```
第1集
 username1：评论内容
  >>user2->user1：回复内容
 username2：评论内容

第2集
 ...
```

- 以 `第N集` 分隔不同剧集
- 评论格式为 `用户名：评论内容`
- 回复评论以 `>>` 开头，格式为 `>>回复者->被回复者：回复内容`

## 数据统计

|------|------|
| 年份跨度 | 1979 - 2025 |

## 使用说明

克隆仓库后即可使用：

```bash
git clone https://github.com/bydbot/bangumi_eposides_comments.git
```

评论文件位于 `comments_text/` 目录下，按年份/季度/番剧ID组织。
